#  Olympic Analytics Dashboard

An interactive data analysis dashboard built with Streamlit that explores more than 120 years of Olympic history.

#Features

- Overall Olympic analysis
- Medal analysis
- Country-wise analysis
- Athlete analysis
- Sports analysis
- Interactive filters
- Country comparison
- Trend visualizations

#Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib

#Dataset

- Olympic Athletes Dataset
- NOC Regions Dataset

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
